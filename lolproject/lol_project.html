League of Legends Project

Data analysis on ranked matches: which factors lead to victory?

Project Overview

This project collects data from the Riot Games API, stores it in MySQL, and analyzes which metrics correlate with winning games. Metrics include Kills, Deaths, Assists, CS, Gold Earned, and more.

Technologies Used

  • Python (requests, pandas, matplotlib)
  • MySQL for data storage
  • Riot Games API for match data
  • GitHub Pages for hosting
